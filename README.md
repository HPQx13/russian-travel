**Цель**

Проект несёт ознакомительную цель - отвечает на вопрос: какие были усвоены и применены технологии веб-разработки на текущей стадии обучения.

Конкретно на текущей стадии обучения использовались технологии:
* grid
* flex
* адаптивная вёрстка
* media запросы
